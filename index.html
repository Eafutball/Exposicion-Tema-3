¡Hola! En esta práctica, aprenderás a obtener datos desde una API utilizando tres métodos diferentes: JQuery Ajax, Fetch y XmlHttpRequest. Cada uno de estos métodos tiene sus propias características y ventajas, lo que te permitirá comprender cómo interactuar con APIs de diversas maneras. Para comenzar, es fundamental que leas las instrucciones detalladas en el documento de la práctica, donde se explican los pasos a seguir y los objetivos que debes alcanzar. Esta práctica te proporcionará una base sólida en la manipulación de datos a través de solicitudes HTTP, ampliando así tus habilidades en el desarrollo web.
